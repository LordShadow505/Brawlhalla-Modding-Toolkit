# Brawlhalla Modding Toolkit

**BMT** is the ultimate multi-tool designed to streamline and enhance the Brawlhalla modding experience. With a comprehensive suite of powerful features, it provides a seamless workflow for creators to modify game assets efficiently.

For detailed guides, tutorials, and instructions on how to use each tool, please visit the [BMT Wiki](https://github.com/LordShadow505/Brawlhalla-Modding-Toolkit/wiki).

---

## Features Overview

### Sprite Exporter
<div align="center">
  <img src="https://github.com/LordShadow505/Brawlhalla-Modding-Toolkit/wiki/Sprite%20Exporter/1.png" alt="Sprite Exporter Interface">
</div>

The Sprite Exporter simplifies and accelerates the modding process by allowing you to easily export Sprites, Shapes, or SWF files. Simplify your modding workflow by isolating specific character skins into clean `.swf` files or exporting raw assets for external editing.

### Reference Loader
<div align="center">
  <img src="https://github.com/LordShadow505/Brawlhalla-Modding-Toolkit/wiki/Reference%20Loader/1.png" alt="Reference Loader Interface">
</div>

The Reference Loader is designed to inject a reference sprite directly into a target `.swf` file, providing a visual guide to ensure your mod proportions and alignments are perfectly accurate.

### Color Converter
<div align="center">
  <img src="https://github.com/LordShadow505/Brawlhalla-Modding-Toolkit/wiki/Color%20Converter/1.png" alt="Color Converter Interface">
</div>

The Color Converter is a straightforward and intuitive tool that supports real-time conversions between multiple color formats (HEX, RGB, INT, HSL, HSV, CMYK). 

### Fast Shape Replacer
<div align="center">
  <img src="https://github.com/LordShadow505/Brawlhalla-Modding-Toolkit/wiki/Fast%20Shape%20Replacer/1.png" alt="Fast Shape Replacer Interface">
</div>

The Fast Shape Replacer automates the tedious process of editing individual shapes by packing all your raw shapes into a single SVG template, letting you replace all the shapes in your folder in one go.

### Skin Editor
<div align="center">
  <img src="https://github.com/LordShadow505/Brawlhalla-Modding-Toolkit/wiki/Skin%20Editor/1.png" alt="Skin Editor Interface">
</div>

The Skin Editor is the ultimate character modding tool. It allows you to interactively manipulate character parts—moving, scaling, and rotating them—all in real-time, powered by a high-performance vector rendering engine.

### Lang Editor
<div align="center">
  <img src="https://github.com/LordShadow505/Brawlhalla-Modding-Toolkit/wiki/Language%20Editor/1.png" alt="Lang Editor Interface">
</div>

The Lang Editor enables you to easily edit almost any text string or language file inside the game, including bulk replacing and instant saving.

### SVG Maker
<div align="center">
  <img src="https://github.com/LordShadow505/Brawlhalla-Modding-Toolkit/wiki/SVG%20Maker/1.png" alt="SVG Maker Interface">
</div>

The SVG Maker is a specialized tool designed to easily convert sprite sheets into highly optimized SVG vectors tailored specifically for generating Brawlhalla assets.
