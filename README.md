# Brawlhalla Modding Toolkit

**BMT** is the ultimate multi-tool designed to streamline and enhance the Brawlhalla modding experience. With a comprehensive suite of powerful features, it provides a seamless workflow for creators to modify game assets efficiently.

For detailed guides, tutorials, and instructions on how to use each tool, please visit the [BMT Wiki](https://github.com/LordShadow505/Brawlhalla-Modding-Toolkit/wiki).

---

## Features Overview

### Sprite Exporter
Easily export Sprites, Shapes, or SWF files. Simplify your modding workflow by isolating specific character skins into clean `.swf` files or exporting raw assets for external editing.

### Reference Loader
Effortlessly load and manage reference images or files to ensure accurate alignment and proportion when designing custom mods or redesigning existing assets.

### Color Converter
Translate and adapt color values (HEX, INT, RGB) to match Brawlhalla's native formats, enabling precise color swapping and palette adjustments.

### Fast Shape Replacer
Streamline the injection of new vector shapes into the game's SWF files. Automate shape replacement to save time when updating complex character assets.

### Skin Editor
A robust, visual interface for modifying character skins in real-time, powered by a high-performance vector rendering engine for infinite-quality SVG previews.

### Lang Editor
Read, parse, and inject modifications directly into the game's text strings (XML/BIN), making language and UI text adjustments seamless.

### SVG Maker
Create and post-process SVG files, optimizing paths and nodes to ensure maximum compatibility and performance when injected back into the game engine.
