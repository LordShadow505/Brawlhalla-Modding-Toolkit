# Color Converter

The Color Converter is a specialized utility for translating and adapting color values (HEX, INT, RGB) to match Brawlhalla's native formats, enabling precise color swapping and palette adjustments.

## Overview

Brawlhalla utilizes specific internal color formats. This tool bridges the gap between standard graphic design color spaces and the game's engine requirements, ensuring that your custom color schemes are applied exactly as intended.
