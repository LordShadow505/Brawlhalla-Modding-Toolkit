# Brawlhalla Modding Toolkit Wiki

Welcome to the official Wiki for the **Brawlhalla Modding Toolkit (BMT)**. This documentation provides comprehensive guides, tutorials, and explanations for every module included in the toolkit.

## Navigation

* **[Sprite Exporter](Sprite_Exporter.md)**: Export Sprites, Shapes, or clean SWF files.
* **[Reference Loader](Reference_Loader.md)**: Manage and load reference images for accurate mod design.
* **[Color Converter](Color_Converter.md)**: Translate color values to match Brawlhalla's native formats.
* **[Fast Shape Replacer](Fast_Shape_Replacer.md)**: Automate the replacement of vector shapes in SWF files.
* **[Skin Editor](Skin_Editor.md)**: Visually modify character skins with high-performance vector rendering.
* **[Lang Editor](Lang_Editor.md)**: Read, parse, and inject text string modifications.
* **[SVG Maker](SVG_Maker.md)**: Optimize paths and nodes for custom SVG creation.
